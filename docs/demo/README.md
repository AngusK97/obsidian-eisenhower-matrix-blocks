# README media demo

This directory contains the reproducible, public demo content used for README screenshots and recordings.

## Set up the demo vault

1. Create a new empty Obsidian vault named `Eisenhower Matrix Blocks Demo`.
2. Install the current local plugin build in `.obsidian/plugins/eisenhower-matrix-blocks/`.
3. Copy `Matrix Demo.md` into the root of the demo vault.
4. Enable Eisenhower Matrix Blocks.
5. Use Obsidian's default theme and disable unrelated community plugins.
6. Open the note in Live Preview or Reading view.
7. Set the plugin interface language to English for the primary README media.

The note contains only fictional launch-planning tasks and is safe to publish. Do not replace it with a note from a personal vault.

## Keep captures reproducible

- Use the same note for light, dark, mobile, and workflow captures.
- Restore `Matrix Demo.md` from this repository before each recording.
- Keep the matrix title and task order unchanged unless the README media plan is updated.
- Hide vault paths, account names, notifications, unrelated tabs, and private file names.
- Capture the actual Obsidian interface. Do not redraw or generate a mock product UI.

See [`../assets/README.md`](../assets/README.md) for exact filenames, dimensions, and capture requirements.
